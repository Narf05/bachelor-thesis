# Figures

This folder contains all figures, plots, and diagrams for the thesis.

## Contents
- Generated plots (from code)
- Manual diagrams
- Charts and visualizations
- Flowcharts and schematics

## Format recommendations
- Use vector formats (PDF, SVG) for LaTeX when possible
- PNG for raster images (300+ DPI for print quality)
- Keep source files (e.g., `.drawio`, `.pptx`) if applicable
